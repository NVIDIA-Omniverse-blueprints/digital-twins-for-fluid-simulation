# kit-streamer

Thin wrapper around the [NVIDIA Omniverse WebRTC Streaming Library](https://www.npmjs.com/package/@nvidia/omniverse-webrtc-streaming-library), built as an IIFE that exposes `window.KitStreamer`.

## What this package produces

Running `npm run build` outputs a single self-contained bundle:

```
trame-app/static/kit-streamer.iife.js
```

That file is generated by the Trame app Docker build and is intentionally not
committed to the repo. If you run the Trame app directly on the host, build it
once with the commands below before starting `trame-app/app.py`.

## When to rebuild

Rebuild when you:

- Bump `@nvidia/omniverse-webrtc-streaming-library` to a new version in `package.json`
- Change `index.js` (the wrapper logic)

No rebuild is needed for changes to the Python trame-app.

## How to rebuild

```bash
cd kit-streamer/
npm install          # only needed after dependency changes
npm run build        # writes trame-app/static/kit-streamer.iife.js
```

Then commit the changed source files:

```bash
git add kit-streamer/package.json kit-streamer/package-lock.json kit-streamer/index.js
git commit -m "chore: update kit-streamer IIFE ..."
```
